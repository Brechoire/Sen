Instructions de configuration

  1. Créez un compte PayPal Developer :

    Rendez-vous sur developer.paypal.com et connectez-vous avec votre compte PayPal.

  2. Créez une application :

    Dans le tableau de bord, cliquez sur "Create App" et remplissez les informations requises.

  3. Récupérez vos identifiants :

    Copiez le "Client ID" et le "Client Secret" de votre application.

  4. Configurez les variables d'environnement :

    Ajoutez ces variables à votre fichier .env ou configurez-les dans votre environnement :

    PAYPAL_CLIENT_ID=votre_client_id_ici
    PAYPAL_CLIENT_SECRET=votre_client_secret_ici
    PAYPAL_MODE=sandbox
  5. Redémarrez le serveur :

    Après avoir configuré les variables, redémarrez votre serveur Django.
