Pour restaurer une sauvegarde :

  1. Téléchargez le fichier ZIP de la sauvegarde souhaitée
  2. Extrayez le contenu du ZIP
  3. Remplacez le fichier db.sqlite3 actuel par celui de la sauvegarde
  4. Extrayez le dossier media/ pour restaurer les images et fichiers
  5. Redémarrez le serveur si nécessaire

⚠️ Attention : La restauration écrasera les données actuelles !
